# 中学校でのセミナー記録

年に一回ですが、茅ヶ崎の中学校でコミュニティセミナーが開催されていました。

その時にプログラミングの講師をしていました。

コロナになる前はパソコンルームにWindows PCがあってツールのインストールが自由だったので、DojaとかActive Rubyとか入れて、好きな言語でやっていましたが、途中から一般的?なJavascriptの方が良いかなと変更しました。

コロナ後は生徒の皆さん一人一人がiPadを持てるようになったのでSwift - PlayGroundでやっています。(パソコンルームは廃止のまま)

| 年度 | 言語、ツール |
| -- | -- |
| 2022-2025 | Swift、playground |
| 2020-2021 | コロナで中止 |
| 2015-2019 | HTML+Javascript、ブラウザで表示 |
| 2009-2014 | Ruby+Tcl/Tk 、ActiveRuby |
| 2005?-2008 | Java、Doja(iアプリ) |
